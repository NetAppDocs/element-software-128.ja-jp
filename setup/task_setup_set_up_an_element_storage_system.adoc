= エレメントストレージシステムをセットアップする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Elementソフトウェア ストレージ システムをセットアップするには、ストレージ ノードの構成、ストレージ クラスターの作成、クラスターへのドライブの追加が必要です。ファイバー チャネル ネットワークを使用する場合は、ファイバー チャネル ノードを構成できます。
