= アクセスグループを削除する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アクセス グループが不要になったら削除できます。グループを削除する前に、ボリューム アクセス グループからイニシエーター ID とボリューム ID を削除する必要はありません。アクセス グループを削除すると、ボリュームへのグループ アクセスは中止されます。

. *管理* > *アクセス グループ*をクリックします。
. 削除するアクセス グループの *アクション* アイコンをクリックします。
. 表示されたメニューで、[削除] をクリックします。
. このアクセス グループに関連付けられているイニシエーターも削除するには、[*このアクセス グループ内のイニシエーターを削除*] チェック ボックスをオンにします。
. 操作を確定します。
