= SnapMirrorを使用した災害復旧
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Elementソフトウェアを実行しているボリュームまたはクラスタに問題が発生した場合、 SnapMirror機能を使用して関係を解除し、宛先ボリュームにフェイルオーバーします。


NOTE: 元のクラスターが完全に失敗したか存在しない場合は、 NetAppサポートに問い合わせてさらにサポートを受けてください。
