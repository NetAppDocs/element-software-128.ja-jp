= 仮想ネットワークを管理する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SolidFireストレージの仮想ネットワークにより、別々の論理ネットワーク上にある複数のクライアント間のトラフィックを 1 つのクラスターに接続できるようになります。クラスタへの各接続は、VLANタギングを使用してネットワーク スタック内で分離されます。
