= ファイバーチャネルセッションを表示する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスターに接続されているファイバー チャネル (FC) セッションを表示できます。ウィンドウに表示する接続のみを含めるように情報をフィルタリングできます。

. Element UI で、*レポート* > *FC セッション* を選択します。
. フィルター条件フィールドを表示するには、[*フィルター*] をクリックします。
