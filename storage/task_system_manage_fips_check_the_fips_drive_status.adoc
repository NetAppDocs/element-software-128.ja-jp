= FIPSドライブのステータスを確認する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Elementソフトウェアを使用して、クラスタでFIPSドライブ機能が有効になっているかどうかを確認できます。 `GetFeatureStatus` FIPS ドライブの有効化ステータスが true か false かを示す API メソッド。

. Element API を使用して、次のように入力してクラスター上の FIPS ドライブ機能を確認します。
+
`GetFeatureStatus`

. 結果を確認する `GetFeatureStatus`API 呼び出し。  FIPS ドライブ有効値が True の場合、FIPS ドライブ機能は有効です。
+
[listing]
----
{"enabled": true,
"feature": "FipsDrives"
}
----
