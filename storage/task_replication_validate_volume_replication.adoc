= ボリュームレプリケーションの検証
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ボリュームが複製された後、ソース ボリュームとターゲット ボリュームがアクティブであることを確認する必要があります。アクティブ状態の場合、ボリュームはペアリングされ、データはソース ボリュームからターゲット ボリュームに送信され、データは同期されています。

. 両方のクラスターから、[*データ保護*] > [*ボリューム ペア*] を選択します。
. ボリュームのステータスがアクティブであることを確認します。
