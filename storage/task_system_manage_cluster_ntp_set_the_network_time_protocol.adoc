= ネットワークタイムプロトコルを設定する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ネットワーク タイム プロトコル (NTP) の設定は、クラスター内の各ノードにブロードキャストをリッスンするように指示するか、各ノードに NTP サーバーに更新を照会するように指示するかのいずれかの方法で実行できます。

NTP はネットワーク上で時計を同期するために使用されます。内部または外部の NTP サーバーへの接続は、初期のクラスター セットアップの一部である必要があります。
