= 保存時の暗号化を有効にする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
保存時にクラスター全体の暗号化を有効または無効にすることができます。この機能は、デフォルトでは有効になっていません。  FIPS ドライブをサポートするには、保存時の暗号化を有効にする必要があります。

. NetApp Elementソフトウェア UI で、*クラスタ* > *設定* をクリックします。
. *保存時の暗号化を有効にする*をクリックします。
