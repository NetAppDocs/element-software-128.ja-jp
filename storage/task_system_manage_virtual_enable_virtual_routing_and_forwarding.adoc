= 仮想ルーティングと転送を有効にする
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
仮想ルーティングおよび転送 (VRF) を有効にすると、ルーティング テーブルの複数のインスタンスをルータに存在させ、同時に動作させることができます。この機能はストレージ ネットワークでのみ使用できます。

VRF を有効にできるのは、VLAN を作成するときだけです。非 VRF に戻す場合は、VLAN を削除して再作成する必要があります。

. *クラスター* > *ネットワーク*をクリックします。
. 新しい VLAN で VRF を有効にするには、[VLAN の作成] を選択します。
+
.. 新しい VRF/VLAN の関連情報を入力します。仮想ネットワークの追加を参照してください。
.. *VRF を有効にする* チェックボックスを選択します。
.. *オプション*: ゲートウェイを入力します。


. *VLAN の作成*をクリックします。
