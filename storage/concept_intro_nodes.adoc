= ノード
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ノードは、ブロック ストレージとしてアクセスできるようにクラスターにグループ化された個々のハードウェア コンポーネントです。  SolidFireストレージ システムには、ストレージ ノードとファイバー チャネル ノードの 2 種類のノードがあります。
