= フィルターを使用して結果を制限する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element UI でページ上のリスト情報をフィルタリングできます。リスト (ボリュームやスナップショットなど) を表示するときに、1 つ以上のフィルターを追加して情報を絞り込み、ページに簡単に収まるようにすることができます。

. リスト情報を表示するときは、[フィルター] をクリックします。
. *フィルター条件*フィールドを展開します。
. フィールドの左端の要素からフィルタリングする列を選択します。
. 列の制約を選択します。
. フィルタリングするテキストを入力します。
. *[追加]*をクリックします。
+
システムはリスト内の情報に対して新しいフィルターを実行し、新しいフィルターを「フィルター条件」フィールドに一時的に保存します。

. 別のフィルターを追加するには、「*追加*」をクリックし、再度フィルターを選択します。
. [すべてクリア] をクリックすると、フィルターのリストが削除され、フィルターされていないリストの情報が表示されます。
