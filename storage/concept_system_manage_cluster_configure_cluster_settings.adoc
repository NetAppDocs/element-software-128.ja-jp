= クラスター設定を構成する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element UI の [クラスター] タブから、クラスター全体の設定を表示および変更したり、クラスター固有のタスクを実行したりできます。

クラスターの完全しきい値、サポート アクセス、保存時の暗号化、仮想ボリューム、 SnapMirror、NTP ブロードキャスト クライアントなどの設定を構成できます。
