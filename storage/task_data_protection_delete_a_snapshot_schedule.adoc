= スナップショットスケジュールを削除する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
スナップショットスケジュールを削除できます。スケジュールを削除すると、今後スケジュールされたスナップショットは実行されなくなります。スケジュールによって作成されたスナップショットはストレージ システムに残ります。

. *データ保護* > *スケジュール*をクリックします。
. 削除したいスケジュールの*アクション*アイコンをクリックします。
. 表示されたメニューで、[削除] をクリックします。
. 操作を確定します。
