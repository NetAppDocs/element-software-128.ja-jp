= iSCSIセッションの詳細
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスターに接続されている iSCSI セッションに関する情報を表示できます。

次のリストは、iSCSI セッションに関して確認できる情報を示しています。

* *ノード*
+
ボリュームのプライマリ メタデータ パーティションをホストするノード。

* *アカウント*
+
ボリュームを所有するアカウントの名前。値が空白の場合、ダッシュ (-) が表示されます。

* *音量*
+
ノード上で識別されるボリューム名。

* *ボリュームID*
+
ターゲット IQN に関連付けられているボリュームの ID。

* *イニシエーターID*
+
イニシエーターのシステム生成 ID。

* *イニシエーターエイリアス*
+
長いリスト内でイニシエーターを簡単に見つけられるようにする、イニシエーターのオプションの名前。

* *イニシエーターIP*
+
セッションを開始するエンドポイントの IP アドレス。

* *イニシエーターIQN*
+
セッションを開始するエンドポイントの IQN。

* *ターゲットIP*
+
ボリュームをホストしているノードの IP アドレス。

* *ターゲットIQN*
+
ボリュームのIQN。

* *チャップ*
+
iSCSI セッションの CHAP アルゴリズム。  CHAP アルゴリズムが使用されていない場合は、ダッシュ (-) が表示されます。  Element 12.8 以降で使用可能です。

* *作成日*
+
セッションが確立された日付。
