= ボリュームパフォーマンスを表示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスター内のすべてのボリュームの詳細なパフォーマンス情報を表示できます。情報は、ボリューム ID またはパフォーマンス列のいずれかで並べ替えることができます。特定の基準で情報をフィルタリングすることもできます。

*[更新間隔]* リストをクリックして別の値を選択すると、ページ上のパフォーマンス情報を更新する頻度を変更できます。クラスターのボリューム数が 1000 未満の場合、デフォルトの更新間隔は 10 秒です。それ以外の場合、デフォルトは 60 秒です。  「しない」の値を選択した場合、自動ページ更新は無効になります。

*自動更新をオンにする*をクリックすると、自動更新を再度有効にすることができます。

. Element UI で、*レポート* > *ボリューム パフォーマンス* を選択します。
. ボリューム リストで、ボリュームの [アクション] アイコンをクリックします。
. *詳細を表示*をクリックします。
+
ボリュームに関する一般情報を含むトレイがページの下部に表示されます。

. ボリュームの詳細情報を表示するには、[詳細を表示] をクリックします。
+
システムはボリュームの詳細情報とパフォーマンス グラフを表示します。
