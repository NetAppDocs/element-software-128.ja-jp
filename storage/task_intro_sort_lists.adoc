= リストを並べ替える
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element UI の特定のページで、リスト情報を 1 つ以上の基準で並べ替えることができます。これにより、画面上で必要な情報を整理することができます。

. 単一の列で並べ替えるには、情報が希望の順序で並べ替えられるまで列見出しをクリックします。
. 複数の列を使用して並べ替えるには、各列の情報が希望の順序で並べ替えられるまで、並べ替える各列の列見出しをクリックします。
+
複数の列を使用して並べ替える場合は、[並べ替え] ボタンが表示されます。

. 並べ替え条件を並べ替えるには、次の手順を実行します。
+
.. *並べ替え*をクリックします。
+
システムは、選択した列に基づいて「並べ替え」フィールドを入力します。

.. リストを並べ替える順序に従って、[並べ替え] フィールドの列を並べ替えます。
+
システムはリスト情報をソートします。



. 単一の並べ替え基準を削除するには、並べ替え基準の名前の横にある [*削除*] アイコンをクリックします。
. すべての並べ替え条件を削除するには、[すべてクリア] をクリックします。
