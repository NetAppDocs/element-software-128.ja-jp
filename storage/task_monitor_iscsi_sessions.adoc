= iSCSIセッションの表示
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスターに接続されている iSCSI セッションを表示できます。必要なセッションのみを含めるように情報をフィルタリングできます。

. Element UI で、*レポート* > *iSCSI セッション* を選択します。
. フィルター条件フィールドを表示するには、[*フィルター*] をクリックします。
