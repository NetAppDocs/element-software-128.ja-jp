= ファイバーチャネルセッションの詳細
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クラスターに接続されているアクティブなファイバー チャネル (FC) セッションに関する情報を確認できます。

次のリストは、クラスターに接続された FC セッションについて確認できる情報を示しています。

* *ノードID*
+
接続のセッションをホストするノード。

* *ノード名*
+
システムによって生成されたノード名。

* *イニシエーターID*
+
イニシエーターのシステム生成 ID。

* *イニシエーターWWPN*
+
開始ワールドワイド ポート名。

* *イニシエーターエイリアス*
+
長いリスト内でイニシエーターを簡単に見つけられるようにする、イニシエーターのオプションの名前。

* *ターゲットWWPN*
+
対象となるワールドワイド ポート名。

* *ボリューム アクセス グループ*
+
セッションが属するボリューム アクセス グループの名前。

* *ボリューム アクセス グループ ID*
+
アクセス グループのシステム生成 ID。
