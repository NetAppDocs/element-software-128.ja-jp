= アカウントの詳細
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
[管理] タブの [アカウント] ページには、各アカウントに割り当てられたボリュームの ID、ユーザー名、効率の詳細など、システム内の各アカウントに関する情報が表示されます。

* *ID*
+
アカウントのシステム生成 ID。

* *ユーザー名*
+
アカウントの作成時に付けられた名前。

* *状態*
+
アカウントのステータス。有効な値は次のとおりです。

+
** アクティブ: アクティブなアカウント。
** ロック済み: ロックされたアカウント。
** 削除済み: 削除され、消去されたアカウント。


* *アクティブボリューム*
+
アカウントに割り当てられたアクティブなボリュームの数。

* *圧縮*
+
アカウントに割り当てられたボリュームの圧縮効率スコア。

* *重複排除*
+
アカウントに割り当てられたボリュームの重複排除効率スコア。

* *シンプロビジョニング*
+
アカウントに割り当てられたボリュームのシンプロビジョニング効率スコア。

* *全体的な効率*
+
アカウントに割り当てられたボリュームの全体的な効率スコア。
