= アクティブペアボリュームのリスト
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListActivePairedVolumes`ボリュームとペアになっているすべてのアクティブボリュームを一覧表示するメソッド。このメソッドは、アクティブなペアリングと保留中のペアリングを持つボリュームに関する情報を返します。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
ボリューム
 a| 
ペアになっているボリュームのボリューム情報。
 a| 
xref:reference_element_api_volumepair.adoc[ボリュームペア]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListActivePairedVolumes",
   "params": {
      },
   "id" : 1
}
----
