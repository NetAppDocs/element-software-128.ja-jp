= クラスター情報を取得
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `GetClusterInfo`クラスターに関する構成情報を返すメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
クラスター情報
 a| 
クラスター情報。
 a| 
xref:reference_element_api_clusterinfo.adoc[クラスター情報]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "GetClusterInfo",
   "params": {},
   "id" : 1
}
----
