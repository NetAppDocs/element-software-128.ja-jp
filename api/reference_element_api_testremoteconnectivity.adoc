= リモート接続のテスト
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用できます `TestRemoteConnectivity`リモート クラスターの各ノードを ping し、リモート アンサンブル データベース接続を確認する方法。このメソッドで有用な結果を返すには、クラスターをペアにする必要があります。リモート データベース接続が失敗した場合、システムからの応答に例外がリストされます。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
詳細
 a| 
各ノードの個別の ping 応答時間。
 a| 
JSONオブジェクト

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "method": "TestRemoteConnectivity",
    "params": {
      "force": "true"
    },
    "id": 1
}
----
