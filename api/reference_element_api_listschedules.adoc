= リストスケジュール
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用できます `ListSchedules`作成されたすべてのスケジュールされたスナップショットに関する情報を取得します。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
schedules
 a| 
現在クラスター上にあるスケジュールのリスト。
 a| 
xref:reference_element_api_schedule.adoc[スケジュール]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "method": "ListSchedules",
    "params": {},
"id": 1
}
----
