= ドライブ統計を取得
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `GetDriveStats`単一のドライブの高レベルのアクティビティ測定値を取得する方法。値は、ドライブをクラスターに追加してから累積されます。一部の値はブロック ドライブに固有です。このメソッドを実行すると、ブロック ドライブ タイプまたはメタデータ ドライブ タイプのいずれかの統計データが返されます。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
ドライブID
 a| 
要求されたドライブの ID。
 a| 
integer
 a| 
なし
 a| 
はい

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
ドライブ統計
 a| 
指定されたドライブIDのドライブアクティビティ情報。
 a| 
xref:reference_element_api_drivestats.adoc[ドライブ統計]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "GetDriveStats",
   "params": {
     "driveID": 3
   },
   "id" : 1
}
----
