= ボリュームアクセスグループにボリュームを追加
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `AddVolumesToVolumeAccessGroup`指定されたボリューム アクセス グループにボリュームを追加する方法。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
ボリューム
 a| 
ボリューム アクセス グループに追加するボリューム ID のリスト。
 a| 
整数配列
 a| 
なし
 a| 
はい



 a| 
ボリュームアクセスグループID
 a| 
ボリュームが追加されるボリューム アクセス グループの VolumeAccessGroupID。
 a| 
integer
 a| 
なし
 a| 
はい

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===


| Name | 説明 | タイプ 


 a| 
ボリュームアクセスグループ
 a| 
新しく変更されたボリューム アクセス グループに関する情報を含むオブジェクト。
 a| 
xref:reference_element_api_volumeaccessgroup.adoc[ボリュームアクセスグループ]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "AddVolumesToVolumeAccessGroup",
   "params": {
      "volumeAccessGroupID": 96,
      "volumes": [1,2]
   },
   "id": 1
}
----
