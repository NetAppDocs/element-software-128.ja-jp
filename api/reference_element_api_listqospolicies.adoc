= QoSポリシーの一覧
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListQoSPolicies`システム上のすべての QoS ポリシーの設定を一覧表示するメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドには次の戻り値があります。

|===
| Name | 説明 | タイプ 


 a| 
QoSポリシー
 a| 
各 QoS ポリシーの詳細のリスト。
 a| 
xref:reference_element_api_qospolicy.adoc[QoSポリシー]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "id": 231,
  "method": "ListQoSPolicies",
  "params": {}
}
----
