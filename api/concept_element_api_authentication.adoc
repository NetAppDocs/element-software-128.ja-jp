= API認証
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
すべての API リクエストに HTTP 基本認証ヘッダーを含めることで、API を使用するときにシステムで認証できます。認証情報を省略すると、システムは認証されていない要求を HTTP 401 応答で拒否します。システムは TLS 経由の HTTP 基本認証をサポートしています。

API 認証にはクラスター管理者アカウントを使用します。
