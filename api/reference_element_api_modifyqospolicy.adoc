= QoSポリシーの変更
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ModifyQoSPolicy`システム上の既存の QoS ポリシーを変更する方法。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===


| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
qosポリシーID
 a| 
変更するポリシーの ID。
 a| 
integer
 a| 
なし
 a| 
はい



 a| 
名前
 a| 
指定すると、QoS ポリシーの名前 (例: gold、platinum、silver) がこの値に変更されます。
 a| 
string
 a| 
なし
 a| 
いいえ



 a| 
qos
 a| 
指定すると、このポリシーの QoS 設定がこれらの設定に変更されます。部分的な QoS 値を指定し、一部の QoS 設定のみを変更できます。
 a| 
QoSオブジェクト
 a| 
なし
 a| 
いいえ

|===


== 戻り値

このメソッドには次の戻り値があります。

|===


| Name | 説明 | タイプ 


 a| 
QoSポリシー
 a| 
新しく変更された QoS ポリシーの詳細。
 a| 
xref:reference_element_api_qospolicy.adoc[QoSポリシー]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "id": 1950,
  "method": "ModifyQoSPolicy",
  "params": {
    "qosPolicyID": 2,
    "qos": {
      "minIOPS": 51,
      "maxIOPS": 15002,
      "burstIOPS": 15002
    }
  }
}
----
