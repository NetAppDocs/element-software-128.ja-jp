= リセットドライブ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ResetDrives`ドライブを事前に初期化し、現在ドライブ上にあるすべてのデータを削除する方法。ドライブは既存のノードで再利用したり、アップグレードされたノードで使用したりできます。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
ドライブ
 a| 
リセットするデバイス名 (ドライブ ID ではない) のリスト。
 a| 
string
 a| 
なし
 a| 
はい



 a| 
力
 a| 
ドライブをリセットするには true に設定します。
 a| 
ブーリアン
 a| 
なし
 a| 
はい

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
詳細
 a| 
リセットされるドライブの詳細。
 a| 
JSONオブジェクト配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ResetDrives",
   "params": {
      "drives" : "slot3",
      "force" : true
   },
   "id" : 1
}
----

このメソッドは、次の例のような応答を返します。

{

  "result": {
   "details": {

    }
   ]
  },
  "duration": "00:00:28.501269",
  "result": "Passed"
  }
}
----
