= ModifySnapMirrorEndpoint（管理対象外）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
エレメントソフトウェアはこのバージョンの `ModifySnapMirrorEndpoint`管理されていないSnapMirrorエンドポイントのストレージ クラスター名または IP アドレス属性を変更する方法。管理されていないエンドポイントは、Element SnapMirror API を使用して管理することはできません。これらは、 ONTAP管理ソフトウェアまたは API を使用して管理する必要があります。

 a|

 a|

 a|

 a|

 a|

 a|

 a|

|===
