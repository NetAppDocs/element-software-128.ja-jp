= ハードウェア情報を取得する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `GetHardwareInfo`単一ノードのライブハードウェア情報とステータスを取得する方法。ハードウェア情報には通常、製造元、ベンダー、バージョン、ドライブ、その他の関連する識別情報が含まれます。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
力
 a| 
クラスター内のすべてのノードで実行するには、この「force」パラメータを true に設定します。
 a| 
ブーリアン
 a| 
false
 a| 
いいえ

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
ハードウェア情報
 a| 
ノードのハードウェア情報。
 a| 
xref:reference_element_api_hardwareinfo.adoc[ハードウェア情報]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "GetHardwareInfo",
   "params": {
   },
"id" : 1
}
----
