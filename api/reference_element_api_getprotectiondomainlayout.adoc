= 保護ドメインレイアウトの取得
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `GetProtectionDomainLayout`各ノードがどのシャーシとどのカスタム保護ドメインに属しているかを含む、クラスターのすべての保護ドメイン情報を返すメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
保護ドメインレイアウト
 a| 
各ノードとそれに関連付けられた保護ドメインのリスト。
 a| 
JSONリストlink:reference_element_api_nodeprotectiondomains.html["ノード保護ドメイン"]オブジェクト。

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "method": "GetProtectionDomainLayout",
  "params": {},
  "id" : 1
}
----
