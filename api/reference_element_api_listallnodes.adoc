= すべてのノードをリスト
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListAllNodes`クラスター内のアクティブなノードと保留中のノードを一覧表示するメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドには次の戻り値があります。

|===
| Name | 説明 | タイプ 


 a| 
ノード
 a| 
クラスター内のアクティブなノードを記述するオブジェクトのリスト。
 a| 
xref:reference_element_api_node.adoc[ノード]



 a| 
保留中のアクティブノード
 a| 
クラスターの保留中のアクティブ ノードを記述するオブジェクトのリスト。
 a| 
xref:reference_element_api_pendingactivenode.adoc[保留中のアクティブノード]配列



 a| 
保留中のノード
 a| 
クラスターの保留中のノードを記述するオブジェクトのリスト。
 a| 
xref:reference_element_api_pendingnode.adoc[保留中のノード]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListAllNodes",
   "params": {},
   "id" : 1
}
----
