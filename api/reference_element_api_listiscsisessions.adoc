= ISCSIセッションのリスト
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListISCSISessions`クラスター内のボリュームの iSCSI 接続情報を一覧表示するメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
セッション
 a| 
各 iSCSI セッションに関する情報。
 a| 
xref:reference_element_api_session_iscsi.adoc[セッション]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListISCSISessions",
   "params": {},
   "id" : 1
}
----
