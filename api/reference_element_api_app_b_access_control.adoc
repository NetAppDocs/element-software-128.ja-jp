= アクセス制御
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
利用できる Element API メソッドは、設定したアクセスの種類によって異なります。


NOTE: Element UI では利用できない、API 経由で利用できるアクセス タイプがあります。



== アカウント

 a| 
アカウントを削除

|===


== administrator
