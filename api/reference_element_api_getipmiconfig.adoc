= GetIpmiConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `GetIpmiConfig`ノード内のセンサーからハードウェア センサー情報を取得するメソッド。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ 


 a| 
シャーシタイプ
 a| 
各ノード シャーシ タイプの情報を表示するために使用されます。有効な値は次のとおりです。

* all: 各シャーシ タイプのセンサー情報を返します。
* {シャーシ タイプ}: 指定されたシャーシ タイプのセンサー情報を返します。

 a| 
string

|===


== 戻り値

このメソッドには次の戻り値があります。

|===
| Name | 説明 | タイプ 


 a| 
センサー名
 a| 
見つかったセンサーの名前。
 a| 
string



 a| 
ユニークセンサーID
 a| 
センサーの一意の識別子。
 a| 
string

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "method": "GetIpmiConfig",
  "params": {
     "chassisType"; "all"
     },
  "id" : 1
}
----
