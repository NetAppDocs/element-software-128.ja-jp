= ListActiveNodes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListActiveNodes`クラスター内の現在アクティブなノードのリストを返すメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
ノード
 a| 
クラスター内のアクティブなノードのリスト。
 a| 
xref:reference_element_api_node.adoc[ノード]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListActiveNodes",
   "params": {},
   "id" : 1
}
----
