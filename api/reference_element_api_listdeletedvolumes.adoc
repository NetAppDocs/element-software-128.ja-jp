= 削除されたボリュームの一覧
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListDeletedVolumes`削除対象としてマークされ、システムから消去されたボリュームのリストを取得する方法。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
仮想ボリュームを含める
 a| 
デフォルトでは、仮想ボリュームが応答に含まれます。仮想ボリュームを除外するには、false に設定します。
 a| 
ブーリアン
 a| 
true
 a| 
いいえ

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
ボリューム
 a| 
削除されたボリュームのリスト。
 a| 
xref:reference_element_api_volume.adoc[音量]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListDeletedVolumes",
   "params": {},
   "id" : 1
}
----
