= テストアドレスの可用性
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `TestAddressAvailability`ストレージ クラスター内のインターフェイスで特定の IP アドレスが使用されているかどうかを確認する方法。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
インタフェース
 a| 
ターゲット ネットワーク インターフェイス (eth0、Bond10G など)。
 a| 
string
 a| 
なし
 a| 
はい



 a| 
address
 a| 
ターゲット インターフェイスでスキャンする IP アドレス。
 a| 
string
 a| 
なし
 a| 
はい



 a| 
仮想ネットワークタグ
 a| 
ターゲット VLAN ID。
 a| 
integer
 a| 
なし
 a| 
いいえ



 a| 
timeout
 a| 
ターゲット アドレスをテストするためのタイムアウト (秒)。
 a| 
integer
 a| 
5
 a| 
いいえ

|===


== 戻り値

このメソッドには次の戻り値があります。

|===


| Name | 説明 | タイプ 


 a| 
address
 a| 
テストされた IP アドレス。
 a| 
string



 a| 
使用可能
 a| 
要求された IP アドレスが使用中の場合は true、そうでない場合は false です。
 a| 
ブーリアン

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
	"method": "TestAddressAvailability",
	"params": {
		"interface": "Bond10G",
		"address": "10.0.0.1",
		"virtualNetworkTag": 1234
	}
}
----


== 応答例

このメソッドは、次の例のような応答を返します。

[listing]
----
{
	"id": 1,
	"result": {
		"address":"10.0.0.1",
		"available": true
	}
}
----
