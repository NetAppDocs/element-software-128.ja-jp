= ネットワーク構成の取得
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `GetNetworkConfig`ノードのネットワーク構成情報を表示するメソッド。



== パラメータ

このメソッドには入力パラメータはありません。



== 戻り値

このメソッドの戻り値は次のとおりです。

|===
| Name | 説明 | タイプ 


 a| 
network
 a| 
ノードの各ネットワーク インターフェイスのネットワーク接続タイプと現在の設定。
 a| 
xref:reference_element_api_network_all_interfaces.adoc[ネットワーク（すべてのインターフェース）]

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "GetNetworkConfig",
   "params": {},
   "id" : 1
}
----
