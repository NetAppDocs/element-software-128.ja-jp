= 仮想ボリュームAPIメソッド
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element ソフトウェア仮想ボリューム API メソッドを使用すると、仮想ボリューム (VVol) を管理できます。これらの API メソッドを使用して、既存の VVol を表示したり、仮想ボリューム ストレージ コンテナーを作成、変更、削除したりできます。これらのメソッドを使用して通常のボリュームを操作することはできませんが、通常のボリューム API メソッドを使用して VVol に関する情報を一覧表示することはできます。

* xref:reference_element_api_createstoragecontainer.adoc[ストレージコンテナの作成]
* xref:reference_element_api_deletestoragecontainers.adoc[ストレージコンテナの削除]
* xref:reference_element_api_getstoragecontainerefficiency.adoc[ストレージコンテナ効率の取得]
* xref:reference_element_api_getvirtualvolumecount.adoc[仮想ボリュームカウントの取得]
* xref:reference_element_api_listprotocolendpoints.adoc[プロトコルエンドポイントのリスト]
* xref:reference_element_api_liststoragecontainers.adoc[ストレージコンテナのリスト]
* xref:reference_element_api_listvirtualvolumebindings.adoc[仮想ボリュームバインディングのリスト]
* xref:reference_element_api_listvirtualvolumehosts.adoc[仮想ボリュームホストの一覧]
* xref:reference_element_api_listvirtualvolumes.adoc[仮想ボリュームの一覧]
* xref:reference_element_api_listvirtualvolumetasks.adoc[仮想ボリュームタスクの一覧]
* xref:reference_element_api_modifystoragecontainer.adoc[ストレージコンテナの変更]
