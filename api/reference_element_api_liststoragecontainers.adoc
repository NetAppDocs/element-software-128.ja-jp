= ストレージコンテナのリスト
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListStorageContainers`システムに認識されているすべての仮想ボリューム ストレージ コンテナーに関する情報を取得する方法。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
ストレージコンテナID
 a| 
情報を取得するストレージ コンテナー ID のリスト。このパラメータを省略すると、メソッドはシステム内のすべてのストレージ コンテナーに関する情報を返します。
 a| 
UUID配列
 a| 
なし
 a| 
いいえ

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===


| Name | 説明 | タイプ 


 a| 
ストレージコンテナ
 a| 
システム内のすべてのストレージ コンテナーに関する情報を含むオブジェクトのリスト。
 a| 
xref:reference_element_api_storagecontainer.adoc[ストレージコンテナ]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListStorageContainers",
   "params": {
      "storageContainerIDs": ["efda8307-b916-4424-979e-658a3f16894d"]
   },
   "id" : 1
}
----


== 応答例

このメソッドは、次の例のような応答を返します。

[listing]
----
{
  "id": 6395,
  "result": {
    "storageContainers": [
      {
        "accountID": 64,
        "initiatorSecret": "EJ:08An1MyNQmL!7",
        "name": "VvolContainer",
        "protocolEndpointType": "SCSI",
        "status": "active",
        "storageContainerID": "efda8307-b916-4424-979e-658a3f16894d",
        "targetSecret": "g38}zWBK%206jQr~",
        "virtualVolumes": []
      }
    ]
  }
}
----
