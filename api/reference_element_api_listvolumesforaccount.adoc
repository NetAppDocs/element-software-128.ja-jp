= アカウントのボリューム一覧
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListVolumesForAccount`アカウントのアクティブなボリュームと（保留中の）削除済みボリュームを一覧表示するメソッド。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
仮想ボリュームを含める
 a| 
仮想ボリュームはデフォルトで応答に含まれます。仮想ボリュームを除外するには、false に設定します。
 a| 
ブーリアン
 a| 
true
 a| 
いいえ



 a| 
accountID
 a| 
このアカウントIDが所有するすべてのボリュームが返されます。
 a| 
integer
 a| 
いいえ
 a| 
はい

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===


| Name | 説明 | タイプ 


 a| 
ボリューム
 a| 
ボリューム情報のリスト。
 a| 
xref:reference_element_api_volume.adoc[音量]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
   "method": "ListVolumesForAccount",
   "params": {
      "accountID"  : 1
   },
   "id" : 1
}
----
