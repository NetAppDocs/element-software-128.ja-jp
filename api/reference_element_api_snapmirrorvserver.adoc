= スナップミラーVサーバー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorVserver オブジェクトには、宛先ONTAPクラスタのストレージ仮想マシン (または Vserver) に関する情報が含まれています。



== オブジェクトメンバー

このオブジェクトには次のメンバーが含まれます。

|===
| Name | 説明 | タイプ 


 a| 
スナップミラーエンドポイントID
 a| 
宛先ONTAPシステムの ID。
 a| 
integer



 a| 
vserverName
 a| 
SVMの名前。
 a| 
string



 a| 
vserverタイプ
 a| 
Vserver のタイプ。有効な値は次のとおりです。

* data
* admin
* システム
* ノード

 a| 
string



 a| 
vserverサブタイプ
 a| 
Vserver のサブタイプ。有効な値は次のとおりです。

* default
* dp_宛先
* data
* 同期ソース
* 同期先

 a| 
string



 a| 
ルートボリューム
 a| 
Vserver のルート ボリューム。
 a| 
string



 a| 
ルートボリュームアグリゲート
 a| 
ルート ボリュームが作成されるアグリゲート。
 a| 
string



 a| 
vserverAggregateInfo
 a| 
snapMirrorVserverAggregateInfo オブジェクトの配列。
 a| 
JSONオブジェクト



 a| 
管理者状態
 a| 
Vserver の詳細な管理状態。有効な値は次のとおりです。

* 実行
* 停止した
* 開始
* 停止
* 初期化中
* 削除

 a| 
string



 a| 
運用状態
 a| 
Vserver の基本的な動作状態。有効な値は次のとおりです。

* 実行
* 停止した

 a| 
string

|===
