= ListIdpConfigurations
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用することができます `ListIdpConfigurations`サードパーティの IdP の構成を一覧表示するメソッド。オプションとして、 `enabledOnly`現在有効になっている IdP 構成を取得するフラグ、または特定の IdP 構成の情報を照会するための IdP メタデータ UUID または IdP 名。



== パラメータ

このメソッドには次の入力パラメータがあります。

|===
| Name | 説明 | タイプ | デフォルト値 | 必須 


 a| 
有効のみ
 a| 
結果をフィルタリングして、現在有効になっている IdP 構成を返します。
 a| 
ブーリアン
 a| 
なし
 a| 
いいえ



 a| 
idp構成ID
 a| 
サードパーティの IdP 構成の UUID。
 a| 
UUID
 a| 
なし
 a| 
いいえ



 a| 
idp名
 a| 
特定の IdP 名の IdP 構成情報を取得します。
 a| 
string
 a| 
なし
 a| 
いいえ

|===


== 戻り値

このメソッドの戻り値は次のとおりです。

|===


| Name | 説明 | タイプ 


 a| 
idpConfigInfos
 a| 
サードパーティの IdP 構成に関する情報。
 a| 
link:reference_element_api_idpconfiginfo.html["idpConfigInfo"]配列

|===


== リクエスト例

このメソッドのリクエストは次の例のようになります。

[listing]
----
{
  "method": "ListIdpConfigurations",
  "params": {}
}
----
