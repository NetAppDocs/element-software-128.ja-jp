= vCenter Server 用NetApp Elementプラグイン
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
NetApp Element Plug-in for vCenter Server は、VMware vSphere インターフェイスへのプラグインを提供し、 NetApp Elementソフトウェアを実行するストレージ クラスターを管理および監視できるようにします。

vCenter ServerのElementプラグインの詳細については、 https://docs.netapp.com/us-en/vcp/index.html["NetApp Element Plug-in for vCenter Server のドキュメント"^] 。
